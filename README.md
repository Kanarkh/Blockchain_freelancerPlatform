# 개요
  * 제작기간 : 설계 5일, 제작 7일
  * 제작 인원 : 4명 ( adnroid 2명, 서버 1명, blockchain 1명)
  * 맡은 역활 : 팀장, blockchain api 설계 및 서버세팅, 체인코드 (golang) 개발
  * 블록체인 : hyperledger fabric (couchDB offchain 연동 포함)

# APP 사용 흐름

  1. 클라이언트와 프리랜서 플랫폼을 통해 연결
  2. 클라이언트가 플랫폼에 돈 입금 ( 에스크로 )
  3. 프리랜서 계약 이행
  4. 클라이언트 계약 내역 확인
  5. 플랫폼 -> 프리랜서 계약금 송금
